How to install Mistral on Microsoft© Word Desktop

Specify the shared folder as a trusted catalog

  1. Open Microsoft© Word.
  2. Choose the File tab, and then choose Options.
  3. Choose Trust Center, and then choose the Trust Center Settings button.
  4. Choose Trusted Add-in Catalogs.
  5. In the Catalog Url box, enter the full network path to the shared folder catalog: \\LNGOXFAPPD009V\Mistral\
  6. Choose Add Catalog.
  7. Select the Show in Menu check box, and then choose OK.



  8. Close the application so your changes will take effect.

Load the add-in

  1. Re-open Microsoft© Word.
  2. Select My Add-ins on the Insert tab of the ribbon.
  3. Choose SHARED FOLDER at the top of the Office Add-ins dialog box.
  4. Select the name of the add-in (“Clausebank”) and choose OK to insert the add-in.

How to install Mistral on Microsoft© Word Online

  1. Open Microsoft© Word Online.
  2. Select Office Add-ins on the Insert tab of the ribbon.
  3. Choose Upload My Add-in in the top-right of the dialog.
  4. Choose Browse…
  5. Enter: https://raw.githubusercontent.com/clausebank/clausebank.github.io/master/clausebank.xml
  6. Choose Upload.
